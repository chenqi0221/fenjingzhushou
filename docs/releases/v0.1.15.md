## 新增
- 新增火山方舟视觉（volcano-vision）供应商，支持 Seedream 4.5 模型
- 智能分镜节点默认使用火山方舟 API 进行图片分析

## 优化
- 火山引擎图片生成 API 端点从 `/images/generate` 改为 `/images/generations`
- 调整所有比例的分辨率映射，满足 Seedream 4.5 最小 3686400 像素要求
- 图片分析时自动压缩大图，避免超出 API 限制
- 分镜生成节点支持递归查找上游上传图片作为参考图

## 修复
- 修复 set_api_key 命令不识别 volcano-vision provider 的问题
- 修复火山引擎参考图不支持 file:// 路径的问题
